# 고도화 로드맵 및 운영 비용

## 현재 프로토타입 한계

- 데이터: 20개 게임 한국어 리뷰 1,918건 (정적 데이터)
- 검색: 단일 쿼리 기반 Top 5 결과
- 분석: 기획안 전체를 하나의 쿼리로 처리
- UI: 단일 페이지, 히스토리 없음

---

## 고도화 방향

### 1단계 - 데이터 확장
- 크롤링 게임 수 20개 → 200개 이상으로 확대
- 정기 크롤링 자동화 (주 1회 신규 리뷰 수집)
- 앱스토어(iOS) 리뷰 추가
- 영어권 리뷰 포함 다국어 지원

### 2단계 - 검색 정확도 개선
- 기획안을 섹션별로 청킹 후 각각 임베딩 → 다중 쿼리 RAG
- 불만 유형별 가중치 적용 (광고피로 > 난이도 등 장르별 차별화)
- 유사 게임 필터링 (장르 매칭 후 검색)
- 검색 결과 리랭킹 (Cross-encoder 적용)

### 3단계 - 기능 확장
- 분석 히스토리 저장 및 비교
- 기획안 버전별 리스크 변화 추적
- 팀 공유 기능 (링크로 리포트 공유)
- PDF 리포트 다운로드
- 경쟁 게임 직접 지정 분석

### 4단계 - 고도화
- 슈퍼센트 내부 출시 게임 데이터 연동
- 출시 후 실제 평점과 사전 리스크 예측 정확도 검증
- 리스크 예측 모델 자체 파인튜닝
- **LangGraph 자가교정 루프 도입**
  - RAG 품질 체크 루프: 검색 결과 유사도가 낮으면 LLM이 쿼리를 자동 수정 후 재검색
  - 리포트 커버리지 체크 루프: 7가지 불만 유형 중 누락된 항목을 감지하면 해당 유형 추가 검색 후 리포트 보완
  - 전제 조건: 데이터 100개 게임 이상 확보 후 도입 권장 (데이터 부족 시 재검색해도 새로운 결과 없음)
