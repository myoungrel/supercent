RAG 시스템이 제대로 동작하는지 샘플 게임 기획 텍스트로 테스트하세요.

## 테스트 시나리오

다음 3가지 샘플 입력으로 테스트:

**케이스 1 - 광고 과다 게임**
```
장르: 하이퍼캐주얼 러너
광고 방식: 스테이지 클리어마다 30초 강제 광고
난이도: 낮음
수익모델: 광고 전용
```

**케이스 2 - 과금 유도 게임**
```
장르: 아이들 클리커
광고 방식: 선택형 광고 (보상형)
난이도: 중간, 특정 구간에서 유료 아이템 없으면 진행 어려움
수익모델: 광고 + IAP
```

**케이스 3 - 콘텐츠 부족 게임**
```
장르: 퍼즐 하이퍼캐주얼
광고 방식: 스테이지 사이 배너
난이도: 낮음, 스테이지 30개
수익모델: 광고 전용
```

## 확인 항목
- 각 케이스에서 관련 불만 패턴이 검색되는지
- 검색 결과의 유사도 점수가 적절한지 (0.7 이상 권장)
- 리스크 리포트가 구체적으로 생성되는지
- 응답 시간이 5초 이내인지

## 결과 출력 형식
각 케이스별로:
- 검색된 불만 패턴 상위 3개
- 유사도 점수
- 생성된 리스크 리포트 요약
- 통과/실패 여부
